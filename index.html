🔬 スコアはどう計算されていますか？ 
【品質スコア計算式】（v8.0改善版）
視覚負荷 = min(100, 残像時間秒 × 10)
  └─ 残像が長いほど視覚野の活動度が高い
認知負荷 = min(100, 注視時間秒 × 3)
  └─ 注視が長いほど前頭葉の注意制御が高い
時間統合 = min(100, log(注視×残像+1) × 15)
  └─ 注視と残像の時間的統合が側頭葉を活性化

品質スコア = (視覚負荷 + 認知負荷 + 時間統合) / 3

【神経科学的根拠】
✓ 視覚刺激→視覚野活性化:主に血流などの変化（機能的充血）を指標にして神経活動を推測している
✓ 眼球運動学習→小脳可塑化: 小脳可塑性が関与する学習（HOKR）と、血管運動の増幅が並行して観察された
✓ 脳血管運動の増幅: 低周波刺激で実証。東北大の資料では、0.25 Hz（4秒に1回）の水平方向に振動する縦縞模様を繰り返し提示すると、視覚刺激の周波数に一致した血管運動が観察され、繰り返しでその振幅が徐々に増幅。また、その同調した血管運動が一次視覚野だけでなく、大脳皮質や小脳の広い領域にも誘導された

★評価の読み方

各脳領域の★の数は、その領域がそのパターンで「どれだけ主役として、または効率的に機能しているか」を示しています。

後頭葉（受容・入力依存）

凝視時間が長いほど、網膜の物理的な疲れ（適応）に頼るため、星が増えます。

凝視時間が短いほど、外部刺激に頼っていないため、星が減ります。

意味： この値が低いほど、脳の自立度が高い（外部刺激に依存していない）ことを示します。

側頭葉（知覚生成・鮮明度）

「短い凝視で長く・鮮明に見える」ほど、星が増えます。

側頭葉（紡錘状回）が、外部刺激がない状態でどれだけ色や形を再現できているかを示します。

意味： この値が高いほど、イメージ処理能力が優秀です。

前頭葉（注意制御・維持）

残像の絶対的な持続時間が長いほど、イメージを意識に留めるために多くの星を必要とします。

30秒以上の維持には、前頭葉と頭頂葉のネットワークによる強いトップダウン信号が必要です。

意味： この値が高いほど、集中力と意識の維持力が優れています。

頭頂葉（空間注意・統合）

残像を「同じ場所」に安定して固定する能力に関わります。

長時間の維持や、非常に短い刺激からの生成時に、空間情報を補完するために働きます。

意味： この値が高いほど、空間的な安定性とイメージの固定力が強いです。

📊 各パターンの解説

2025年の最新の神経科学研究をもとに、脳活動の推定モデルを構築しています。

🔬 脳領域ごとの役割
前頭葉（維持）：視覚情報の維持と注意の持続力
側頭葉（生成）：残像の鮮明さと視覚記憶の生成力
後頭葉（受容）：視覚刺激の受容と処理
🧪 効率比とは？
効率比は、凝視時間に対する残像時間の比率です。
高いほど、少ない刺激で長い残像を生成できる「脳の内部生成能力」が優れています。
例：5秒凝視で30秒残像 → 効率比 6.0倍
💡 トレーニングの目標
初心者：まずは30秒凝視で30秒残像（パターンE）を目指す
中級者：10～15秒凝視で30秒残像（パターンB・C）を目指す
上級者：5秒凝視で30秒残像（パターンA）や、30秒凝視で60秒残像（パターンF）に挑戦

🎯 推奨トレーニングプラン

1週目：基礎固め

目標：5～30秒凝視で30秒残像

毎日1～2セット、凝視時間は短めでOK。

残像が見えることに慣れる段階です。

2週目：効率化

目標：10～30秒凝視で30秒残像

凝視時間を徐々に短くして、残像時間を維持。

脳の内部生成力を高める段階です。

3週目以降：挑戦

目標：5～30秒凝視で30秒残像

効率比6.0倍以上を目指す。

または、30秒凝視で60秒残像に挑戦。

継続のコツ

無理せず、毎日続けることが大切です。

残像が見えにくい日もあるので、焦らずに。

長期的には、脳の可塑性が高まります。

💊 パターン別アドバイス

パターンA～Cの方へ

短時間の凝視で長い残像を生成できる「高効率型」です。

脳の内部生成力が優れているので、30秒凝視で60秒残像に挑戦してみましょう。

パターンD～Eの方へ

標準的な脳活動パターンです。

まずは10～15秒凝視で30秒残像を目指しましょう。

徐々に凝視時間を短くして、効率比を高めていきます。

パターンFの方へ

長時間の凝視と残像を維持できる「持久型」です。

全脳的な持久力が高いので、さらに長時間のトレーニングに挑戦できます。

または、凝視時間を短くして効率比を高める方向も有効です。

残像が見えない方へ

まずは30秒凝視から始めてください。

目を閉じて暗い場所で確認すると、残像が見えやすくなります。

個人差が大きいので、焦らず続けることが大切です。

次のステップ

目標1：凝視時間の短縮

「30秒凝視 → 30秒残像」から「10秒凝視 → 30秒残像」への挑戦、そして「5秒凝視 → 30秒残像」への挑戦

これは側頭葉のイメージ生成力が星5つになることを意味します。

目標2：応用トレーニング

「輪郭がはっきり見えている状態」で終了を判定

毎回同じ基準で測定することで、進歩が正確に追跡できます
